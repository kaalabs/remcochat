# RemcoChat (PLAN)

Last updated: 2026-02-12

RemcoChat is a minimal ChatGPT-like chat UI for local network use (no auth).

## Current roadmap (high level)

- Profiles (create/switch) + per-profile defaults
- Chat UX (streaming, stop, folders, pins, export)
- Profile photo (upload + position + remove)
- LAN-friendly admin gating (for privileged tools)
